Ввод строкового значения.

Это окно появляется, когда для продолжения необходимо ввести строковое значение.

Строковое значение вводится в поле "Значение". Для подтверждения надо нажать кнопку "ОК", для отмены действия надо нажать кнопку "Отмена".
Кнопка "Справка" используется для вызова справки по окну.
