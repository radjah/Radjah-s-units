Настройка разъёмов места.

В этом окне определяются разъёмы для выбранного места. Для каждого разъёма можно определить клеммы.

Структура меню

Элементы:
  • Создать разъём - определение нового разъёма. При выборе этого пункта вызывается окно для ввода имени нового элемента.
  • Создать группу клемм - создание группы клемм для выбранного разъёма. При выборе этого пункта вызывается окно "Добавление группы клемм".
  • Закрыть - закрытие окна и возврат к основному окну программы.
Помощь:
  • Справка - открытие страницы справки по окну.
