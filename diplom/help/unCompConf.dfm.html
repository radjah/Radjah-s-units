Конфигурация испытываемого оборудования.

В этом окне производится составление списка единиц испытываемого оборудования.

Структура меню

Оборудование:
  • Добавить - добавление единицы оборудования. При выборе этого пункта появится окно с запросом имени добавляемой единицы. Это же действие можно производить с помощью кнопки "Добавить".
  • Удалить - удаление выбранной единицы оборудования. При выборе этого пункта появится запрос на подтверждение действия. Это же действие можно производить с помощью кнопки "Удалить".
  • Указать теги - вызов окна определения типов воздействий, которые КПС может оказывать на выбранную единицу оборудования. Это же действие можно производить с помощью кнопки "Теги".
Помощь:
  • Справка - открытие страницы справки по окну.
