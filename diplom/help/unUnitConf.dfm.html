Настройка шкафа.

В этом окне происходит сопоставление тегов воздействий выбранной единице испытываемого оборудования.

В списке "Теги воздействий" перечислены уже ассоциированные теги для выбранного шкафа. В списке "Сигналы по типа" приведено количество клем для каждого типа сигналов, которое необходимо для подключения этого шкафа. В списке "Доступные теги" перечислены все типы воздействий, которые может производить комплекс.
Нужное воздействие можно найти по тегу или названию, для этого используется поле "По тегу" и "По наименованию" соотвественно.

Для добавления тега воздействия в список "Теги воздействий" необходима выбрать нужный тег в списке "Доступные теги" и нажать кнопку "< Добавить", при этом появится запрос на подтверждение действия.
Для удаление тега воздействия из списка "Теги воздействия" необходима выбрать нужный тег в этом списке и нажать кнопку "Удалить >", при этом появится запрос на подтверждение действия.
Кнопка "Справка" используется для вызова справки по окну.
