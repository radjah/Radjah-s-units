Основное окно программы.

В этом этом окне находится схематичное изображение установки комплекса преобразователей сигналов. Через меню доступны все функции программы.

Структура меню

Файл:
  • Составление карты - открытие окна полуавтоматического составления карты клеммных полей.
  • Ручное составление карты - открытие окна ручного составления карты клеммных полей.
  • Выход - выход из программы.
Элементы:
  • Подставка - создание нового элемента "Подставка". При выборе этого пункта вызывается окно для ввода имени нового элемента.
Система:
  • Конфигурация комплекса - открытие окна изменения конфигурации комплекса.
  • Конфигурация оборудования - открытие окна изменения конфигурации испытываемого оборудования.
  • Типы сигналов - открытие окна изменения списка типов сигналов, которые может вырабатывать комплекс.
  • Теги воздействий - открытие окна изменения списка воздействий, которые могут производиться на испытываемое оборудования.
Помощь:
  • Справка - вызов справки по программе.
Контекстное меню элемента "Подставка":
  • Переименовать - вызов окна ввода нового имени.
  • Удалить - удалить элемент и всего его дочерние элементы. При выборе этого пункта появится запрос на подтверждение действия.
  • Создать место - создание нового элемента "Место" на выбранном элементе "Подставка". При выборе этого пункта вызывается окно для ввода имени нового элемента.
Контекстное меню элементе "Место":
  • Переименовать - вызов окна ввода нового имени.
  • Разъёмы - открытие окна настройки выбранного элемента "Место". Так же это окно можно вызвать, щелкнув по кнопке с названием нужного места.
  • Удалить - удалить элемент и всего его дочерние элементы. При выборе этого пункта появится запрос на подтверждение действия.
