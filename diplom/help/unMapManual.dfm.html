Ручное составление карты клеммных полей.

В этом окне производится ручное определение подключений клемм к шкафам испытываемого оборудования.

Структура меню.

Карта:
  • Отчеты:
    • Для печати - подготовка отчета для вывода на печать. Эту же операцию можно произвести с помощью кнопки "Отчет".
    • В Excel - сформировать отчет и вывести его в программе Microsoft Excel. Эту же операцию можно произвести с помощью кнопки "Отчет в Excel".
  • Экспорт в Excel - экспорт карты в Microsoft Excel для дальнейшей работы с ней. Эту же операцию можно произвести с помощью кнопки "Экспорт в Excel".
  • Закрыть - закрытие окна.
Помощь:
  • Справка - вызов справки по окну.
В списке "Оборудование" перечислены все шкафы, для которых определен хотя бы один тег подключения.  В списке "Теги воздействий" перечислены все теги, которые определены для выбранного шкафа. В списке "Места" перечислены все места для которых определена хотя бы одна клемма для подключения. В списке "Клеммы" перечислены все клеммы для выбранного места и с типом сигнала в соответствии с выбранном тегом воздействия в списке "Теги воздействий". В списке "Соединения" перечислены все созданные соединения.
Примечание: теги и клеммы, которые уже входят в состав соединений в списка "Теги воздействий" и "Клеммы" отсуствуют.

Для создания нового соединения необходимо выбрать нужный шкаф и тег в нем, нужное место и клемму. После этого надо нажать кнопку "Добавить >", появится запрос на подтверждение действия.
Для удаления соединения необходимо выбрать нужное соединение и нажать кнопку "< Удалить", появится запрос для подтверждения действия.
