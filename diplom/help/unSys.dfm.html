Конфигурация комплекса.

В этом окне производится настройка конфигурации комплекса. В списке "Сигналы комплекса" перечислены все типы сигналов, для которых определено количество. В списке "Доступные типы сигналов" перечислены все типы сигналов, для которых еще не определено количество.

Для определения количества необходима выбрать тип сигнала из списка "Доступные типы сигналов" и нажать кнопку "< Добавить", при этом появится окно для ввода количества сигналов.
Для удаления сигналов, для которых определено количество, надо выбрать нужный тип из списка "Сигналы комплекса" и нажать кнопку "Удалить >", при этом появится запрос на подтверждение действия.
Кнопка "Справка" используется для вызова справки по окну.
