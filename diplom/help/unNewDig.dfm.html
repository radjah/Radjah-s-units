Ввод числового значения.

Это окно появляется, когда для продолжения требуется ввести числовое значение.
Чесловое значение необходимо ввести в поле "Введите или выберите числовое значение", для подтверждения и закрытия окна используется кнопка кнопка "ОК", для отмены и закрытия окна используется кнопка "Отмена".
Кнопка "Справка" используется для вызова справки по окну.
