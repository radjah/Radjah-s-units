Полуавтоматическое формирование карты клеммных полей.

В этом окне производится автоматический поиск подходящий мест для подключения выбранного шкафа и автоматическое формирование подключений для выбранного шкафа и места.

Структура меню

Карта:
  • Отчеты:
    • Для печати - подготовка отчета для вывода на печать. Эту же операцию можно произвести с помощью кнопки "Отчет".
    • В Excel - сформировать отчет и вывести его в программе Microsoft Excel. Эту же операцию можно произвести с помощью кнопки "Отчет в Excel".
  • Экспорт в Excel - экспорт карты в Microsoft Excel для дальнейшей работы с ней. Эту же операцию можно произвести с помощью кнопки "Экспорт в Excel".
  • Закрыть - закрытие окна.
Помощь:
  • Справка - вызов справки по окну.

В списке "Шкафы" перечисленны доступны для подключения шкафы. В списке "Сигналы для шкафа" перечисленны количества сигналов по типам для выбранного шкафа. В списке "Подходящие места" перечислены найденные места для подключения выбранного шкафа. В списке "Сигналы для месты" перечисленны количества сигналов по типам для выбранного найденного места. В списке "Соединения" перечислены все определеные соединения.

Для подключения шкафа небходимо выбрать нужный шкаф из списка "Шкафы" и нажать кнопку "Найти место >". После этого надо выбрать нужное место и нажать "Подключить >".

Перед началом формирования карты рекомендуется удалить все существующие соединения с помощью кнопки "Очистить".
